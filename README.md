# FideliusCrypto 🔐

A .NET library for secure ECDH-based encryption using **BouncyCastle**. Provides **key exchange, AES-GCM encryption, and HKDF key derivation**.

---

## 📦 **Installation**  
### **NuGet Package**
Install the package via NuGet:  
```sh
dotnet add package FideliusCrypto
